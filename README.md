Sistema de gestión y seguimiento de vehículos.

VehiTrack es una aplicación desarrollada como proyecto de formación del SENA, diseñada para facilitar la administración y el seguimiento de vehículos mediante una interfaz intuitiva y una base de datos relacional.

Tecnologías utilizadas
HTML5
CSS3
JavaScript
Python (si aplica)
MySQL
Git
GitHub
Funcionalidades
Registro de vehículos.
Consulta de información.
Actualización de registros.
Eliminación de registros.
Gestión de la base de datos.
